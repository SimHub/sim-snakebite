## What you need   

  - node js
  - npm 

## USAGE   

#### install 

    clone or download .zip file  

#### setup  

    : cd sim-snakebite
    : npm install  
    : npm run start  


#### development 

    : npm run dev  
    : node server.js // separate terminal
